This is the first post on my new Jekyll blog.

To add a new post, drop a file into `_posts/` named with the pattern
`YYYY-MM-DD-some-slug.markdown`. Jekyll picks it up automatically — the date
in the filename becomes the post date, and the slug becomes the URL.

Markdown works as you'd expect:

- bullet lists,
- **bold** and *italic*,
- and code:

```ruby
puts "Hello, Minima!"
```

You can also use math, images, footnotes — see the
[Jekyll docs](https://jekyllrb.com/docs/posts/) for the full set of features.
